# Changelog

All notable changes to the DCP Provider CLI VS Code extension will be documented in this file.

## [1.0.0] - 2026-03-23

### Added
- Initial release of DCP Provider CLI extension
- Provider node status monitoring (online/offline/error)
- Node management commands (start, stop, restart)
- Real-time CPU and memory usage tracking
- Active job counter and total jobs completed
- GPU allocation management interface
- GPU health monitoring (temperature, allocation %)
- Earnings tracking (total, monthly, pending)
- Workspace configuration system with `.dcp/provider.json`
- Secure API key storage in VS Code SecretStorage
- Status bar indicator with live node health
- Output channel for detailed logs and diagnostics
- Health check polling with configurable interval
- Tree view providers for organized information display
- Keyboard shortcuts for common operations
- Comprehensive error handling and user feedback
- README with full documentation and examples

### Features
- Node Status sidebar view
- GPU Allocation sidebar view
- Earnings sidebar view
- 8 commands for provider management
- Automatic background health monitoring
- Configuration validation
- Secure credential storage
