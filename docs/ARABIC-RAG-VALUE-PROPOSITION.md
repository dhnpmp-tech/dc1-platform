# Arabic RAG-as-a-Service: Value Proposition & Copy

## Product Overview

**One-Click Arabic RAG** is a complete retrieval-augmented generation (RAG) template on DCP that bundles:
- **BGE-M3 Embeddings** (multilingual, optimized for Arabic)
- **BGE Reranker** (Arabic-aware semantic reranking)
- **ALLaM 7B LLM** (ALLaM is Saudi Arabia's national AI model)

Deployed on DCP GPUs, this provides **in-kingdom, PDPL-compliant Arabic document processing** at 40-60% lower cost than hyperscalers.

## PDPL Compliance Positioning

**What is PDPL?**
Saudi Personal Data Protection Law (PDPL, effective 2024) requires:
- Data processing must comply with PDPL standards
- Government and sensitive sectors prefer in-kingdom data handling
- Non-compliance = fines, reputational risk

**DCP's Advantage:**
- All processing happens in Saudi Arabia (no cloud egress)
- No cross-border data transfer agreements needed
- Competitive edge: Competitors using AWS/Azure are at regulatory risk
- Peace of mind: Full PDPL compliance by architecture, not by contract

**Marketing Copy:**
> "DCP's Arabic RAG is PDPL-native. Your data never leaves the Kingdom. Achieve enterprise compliance by design."
