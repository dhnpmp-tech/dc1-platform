## 1. Brand Foundation

### Company
**DCP — Decentralized Compute Platform**
DC Power Solutions Co | CR 7053667775 | Riyadh, Saudi Arabia

### Tagline
> **"Infinite Compute. Real Power."**

### Brand Story
The ∞ infinity symbol is not decoration — it's the business model. DCP connects GPU providers with AI builders in a loop that never ends: energy becomes compute, compute becomes intelligence, intelligence creates value, value powers more energy. Infinite. Real.

- **D** = Decentralized nodes — anyone can provide
- **C** = Compute — raw GPU inference power
- **P** = Power — Saudi energy advantage, real throughput

### Mission
Make GPU compute accessible, affordable, and sovereign for builders everywhere — starting with the Arabic-speaking world.

### Vision
The default compute layer for AI in the Arab world and beyond.

### Values
1. **Open** — OpenAI-compatible API, no walled garden
2. **Sovereign** — Your data stays in your region (PDPL compliant)
3. **Real** — No vaporware. Working inference or nothing.
4. **Inclusive** — Arabic-first, but not Arabic-only

### Logo Rules

**✅ DO:**
- Use on dark navy `#0D1B2A` backgrounds by default
- Maintain clear space equal to the height of the "D" on all sides
- Use the monochrome version on white/light backgrounds
- Use the icon-only ∞ for favicons and small contexts

**❌ DON'T:**
- Recolor the gradient (orange→teal is locked)
- Stretch, rotate, or distort
- Place on busy photographic backgrounds without a dark overlay
- Use any legacy DC1 logos (circuit board, lightning bolt)
- Add drop shadows or additional effects beyond the built-in glow

- Orange for energy/power messaging, warnings, highlights

## 4. Typography

### Primary Typefaces

**Headings / Display — Inter Bold or Geist Bold**
```
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789
```
- Weight: 700 (Bold)
- Use: Headlines, hero text, the DCP wordmark
- Letter-spacing: +2 to +4px for display sizes

**Body — Inter Regular**
- Weight: 400 (Regular) / 500 (Medium) for emphasis
- Use: Body copy, descriptions, UI text
- Line height: 1.6

**Arabic — Cairo**
- Use: ALL Arabic text — never romanize Arabic content
- Weights: Cairo Regular (400), Cairo SemiBold (600), Cairo Bold (700)
- RTL layout is mandatory for Arabic content

**Monospace — JetBrains Mono**
- Use: Code samples, API endpoints, technical specs
- Colour: Teal `#00E5C8` on navy background

### Type Scale
| Level | Size | Weight | Use |
|-------|------|--------|-----|
| Display | 56–72px | Bold | Hero headlines |
| H1 | 40px | Bold | Page titles |
| H2 | 32px | Bold | Section headings |
| H3 | 24px | SemiBold | Subsections |
| Body Large | 18px | Regular | Lead copy |
| Body | 16px | Regular | Standard body |
| Caption | 14px | Regular | Labels, captions |
| Code | 14px | Mono | Code, API |

## 5. Brand Voice

### Character
DCP speaks like a senior engineer who built something real and doesn't need to shout about it.

- **Confident** — We know what we built. No hedging.
- **Technical** — Specifics over generalities. Numbers over adjectives.
- **Direct** — Get to the point. No filler.
- **Warm** — We're building for people, not enterprises.

### Tone by Context

| Context | Tone | Example |
|---------|------|---------|
| Product / API docs | Precise, instructional | "Send a POST request to /v1/chat/completions" |
| Provider onboarding | Encouraging, practical | "Your GPU earns while you sleep" |
| Enterprise sales | Professional, evidence-based | "PDPL-compliant inference. Audit trail included." |
| Social media | Confident, conversational | "Arabic AI just got faster." |
| Error messages | Clear, helpful | "No providers online for this model. Try Llama-3.1-8B." |

### Voice Rules

**✅ DO:**
- Use Arabic script for Arabic content (never romanized)
- Lead with the product, not the company
- State numbers (SAR/hour, tokens/second, VRAM)
- Bilingual = Arabic first, English second for MENA audience

**❌ DON'T:**
- "Excited to announce…"
- "Game-changing" / "revolutionary" / "disruptive"
- "We're on a mission to…" (show it, don't tell it)
- Generic Silicon Valley startup-speak

### Key Messages by Audience

**For Developers:**
> "OpenAI-compatible. Deploy in 60 seconds. Arabic models included."

**For GPU Providers:**
> "Your GPU earns SAR while you sleep. Daemon installs in one command."

**For Enterprise:**
> "PDPL-compliant inference. Your data never leaves the Kingdom."

**For Investors:**
> "Saudi energy + decentralized GPU network + Arabic AI gap = the infrastructure play of Vision 2030."

## 6. Photography & Imagery

### Style
- **Dark-first** — All compositions work on `#0D1B2A` background
- **Technical + Human** — GPU hardware alongside real people, not stock
- **MENA-authentic** — Saudi settings, Arabic script, Middle Eastern representation
- **Data visualization** — Network flows, inference pipelines, benchmark graphs

### What to Use
- GPU hardware (RTX cards, server racks) in warm lighting
- Riyadh/MENA city contexts (not Western datacenter stock)
- Arabic calligraphy rendered as data/light
- Abstract teal and orange data flow visualizations
- Authentic portraits (no stock office photos)

### What to Avoid
- Generic Silicon Valley office imagery
- Blue-and-white color schemes
- Robotic arms (cliché AI imagery)
- White backgrounds
- Western-only representation

## 7. Digital Standards

### Web
- Background: `#0D1B2A` primary, `#1A2F45` cards
- Text: `#FFFFFF` primary, `#A8B8C8` secondary
- CTAs: Teal `#00E5C8` background, `#0D1B2A` text
- Links: Teal `#00E5C8`, hover: `#00B39E`
- Border radius: 8px (cards), 6px (buttons), 4px (inputs)
- Font: Inter (EN), Cairo (AR)

### Email
- Dark background preferred (`#0D1B2A`)
- Logo at top, centered
- Single column, max 600px width
- CTA button: teal with navy text

### Social Media Formats

| Platform | Primary Format | Logo placement |
|----------|---------------|----------------|
| LinkedIn | 1200×627 | Top-left or bottom-left |
| Twitter/X | 1500×500 header, 1200×675 posts | Bottom-left |
| Instagram | 1080×1080 (posts), 1080×1920 (stories) | Bottom-center |
| YouTube | 2560×1440 banner | Top-left |
