## العنوان الأساسي

- عبر البروكسي: `https://dcp.sa/api/dc1`
- مباشرة: `http://76.13.179.86:8083/api`

## المصادقة

- المزود: `?key=` أو `x-provider-key`
- المستأجر: `?key=` أو `x-renter-key`
- المدير: `x-admin-token`
