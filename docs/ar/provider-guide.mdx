## خطوات المزود

1. التسجيل في `/provider/register`
2. تنزيل السكربت أو المثبت
3. تشغيل `dc1_daemon.py` بشكل مستمر
4. متابعة الأرباح والسحب من لوحة المزود

## التثبيت على لينكس

```bash
curl -sL "https://dcp.sa/api/dc1/providers/download/setup?key=<PROVIDER_KEY>&os=linux" | bash
```

## الحصة المالية

- حصة المزود: 75%
- حصة DC1: 25%
