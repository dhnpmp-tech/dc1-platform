## ما هو النموذج

`ALLaM-AI/ALLaM-7B-Instruct-preview` نموذج تعليمات بحجم 7B من SDAIA (السعودية) ومصمم لحالات استخدام عربية ومؤسسية.

## ماذا يفعل

- محادثة عربية رسمية وشبه رسمية.
- تلخيص المحتوى العربي الطويل.
- إنتاج ردود مساعدة للقطاع الحكومي والمالي.

## مقارنة سريعة

- مقابل `Falcon H1 7B`: عادة أفضل في اتساق الأسلوب العربي المحلي.
- مقابل `JAIS 13B`: أقل استهلاكًا للذاكرة، مع جودة جيدة جدًا مقابل تكلفة أقل.

## أفضل استخدام على DCP

- مساعد عربي لخدمة العملاء.
- واجهات داخلية للبحث المؤسسي العربي.
- توليد ردود متوافقة مع نبرة الأعمال.

## متطلبات العتاد

- الحد الأدنى العملي: `24 GB VRAM`.
- GPU مناسب: `RTX 4090` أو `A100 40GB`.

## التشغيل على DCP

- قالب موصى به: `vllm-serve`
- معرف النموذج:

```text
ALLaM-AI/ALLaM-7B-Instruct-preview
```

- مثال طلب API:

```json
{
  "job_type": "vllm_serve",
  "params": {
    "model": "ALLaM-AI/ALLaM-7B-Instruct-preview",
    "max_model_len": 8192,
    "dtype": "float16"
  }
}
```
