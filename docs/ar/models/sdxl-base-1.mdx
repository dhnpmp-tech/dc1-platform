## ما هو النموذج

`stabilityai/stable-diffusion-xl-base-1.0` نموذج Diffusion لتوليد الصور من النصوص بدقة وجودة أعلى من الأجيال الأقدم.

## ماذا يفعل

- إنتاج صور تسويقية وإبداعية من Prompt نصي.
- دعم أنماط متعددة (واقعي، فني، تصميم مفاهيمي).
- مناسب للتجارب السريعة ومخرجات الإنتاج.

## مقارنة سريعة

- مقابل `CompVis/stable-diffusion-v1-4`: SDXL يقدم جودة أعلى وتفاصيل أدق لكنه يحتاج موارد أكبر.

## أفضل استخدام على DCP

- حملات تسويق مرئية.
- توليد صور منتجات أولية.
- محتوى إبداعي عربي/إنجليزي.

## متطلبات العتاد

- الحد الأدنى العملي: `16 GB VRAM`.
- GPU مناسب: `RTX 4090` أو `A100 40GB`.

## التشغيل على DCP

- قالب موصى به: `stable-diffusion`
- معرف النموذج:

```text
stabilityai/stable-diffusion-xl-base-1.0
```

- مثال طلب API:

```json
{
  "job_type": "image_generation",
  "params": {
    "model": "stabilityai/stable-diffusion-xl-base-1.0",
    "prompt": "واجهة مدينة الرياض المستقبلية عند الغروب",
    "steps": 30,
    "width": 1024,
    "height": 1024
  }
}
```

## الترخيص والاستخدام التجاري

- الترخيص: `OpenRAIL++`.
- الاستخدام التجاري: مسموح غالبًا مع الالتزام بسياسات الاستخدام.
