## ما هو النموذج

`inceptionai/jais-13b-chat` نموذج ثنائي اللغة (عربي/إنجليزي) بحجم 13B من Inception بالتعاون مع G42.

## ماذا يفعل

- إجابات عربية عالية الجودة مقارنة بفئة 7B.
- فهم أفضل للسياق العربي الطويل نسبيًا.
- مناسب لوكلاء المعرفة والمساعدة المؤسسية.

## مقارنة سريعة

- مقابل `ALLaM 7B` و`Falcon H1 7B`: عادة جودة أعلى في العربية، لكن بتكلفة أعلى وزمن استجابة أطول.

## أفضل استخدام على DCP

- مساعد معرفي عربي في قطاعات حساسة (تمويل، صحة، حكومة).
- نظام FAQ عربي بجودة لغوية عالية.

## متطلبات العتاد

- الحد الأدنى العملي: `24 GB VRAM`.
- GPU مناسب: `RTX 4090` أو `A100 40GB`.

## التشغيل على DCP

- قالب موصى به: `vllm-serve`
- معرف النموذج:

```text
inceptionai/jais-13b-chat
```

- مثال طلب API:

```json
{
  "job_type": "vllm_serve",
  "params": {
    "model": "inceptionai/jais-13b-chat",
    "max_model_len": 4096,
    "dtype": "float16"
  }
}
```

## الترخيص والاستخدام التجاري

- الترخيص: `Apache 2.0`.
- الاستخدام التجاري: مسموح ضمن شروط الترخيص.
