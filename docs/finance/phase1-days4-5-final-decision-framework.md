## Go/No-Go Decision Matrix

### 🟢 **GREEN LIGHT — GO**
**Approve Phase 2 if ALL criteria met:**
- Revenue ≥ $500 ✓
- ≥3 renters ✓
- ≥5 providers ✓
- Provider margins ≥ $30/month ✓
- Repeat rate ≥ 15% ✓
- CAC payback ≤ 2.5 months ✓
- No critical issues ✓

**Action:** Activate Phase 2, deploy $50-75K budget

---

### 🟡 **YELLOW LIGHT — CONDITIONAL GO**
**Approve Phase 2 with caution if 1-2 criteria below target:**
- Revenue $300-500 (lower but viable)
- 2-3 renters (small but growing)
- 3-5 providers (minimal but adequate)
- Provider margins $20-50/month (workable)
- CAC payback 2.5-3.5 months (tight but acceptable)

**Action:** Approve Phase 2 with $30-40K budget, monthly reviews

---

### 🔴 **RED LIGHT — NO-GO**
**Defer Phase 2 if 2+ criteria fail:**
- Revenue < $300 (insufficient validation)
- <2 renters (no customer base)
- <3 providers (supply side broken)
- Provider margins < $20/month (unsustainable)
- CAC payback > 3.5 months (economics fail)
- Critical technical issues (system unreliable)

**Action:** Enter Phase 1b optimization, re-assess 2026-04-15

---

## Red Flag Overrides

**Automatic NO-GO if ANY of these occur:**
1. Security incident (data breach, payment fraud)
2. Data loss or integrity issue
3. >30% provider churn in single day
4. Regulatory/legal blocker
5. System down >2 hours
