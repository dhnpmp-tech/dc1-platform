**Easiest channel, highest conversion**

**For each persona, run 3 searches and send connection requests:**

Search 2: "Saudi Arabia" + "government technology" + "engineering"
Search 3: "Riyadh" + "finance" + "CTO"

Send: Personalized connection request (copy template from checklist)

**Persona B: Arabic NLP Developer** (Target: 2-3 people)
```
Search 1: "Arabic NLP" + "founder"
Search 2: "Arabic AI" + "engineer"
Search 3: "MENA" + "machine learning" + "startup"

Send: Personalized connection request (copy template from checklist)

**Persona C: Western ML Engineer** (Target: 1-2 people)
```
Search 1: "GPU" + "machine learning engineer" (US/EU filter)
Search 2: "ML Engineer" + "AWS" (US/EU filter)
Search 3: "AI researcher" + "computer vision" (US/EU filter)

**Timeline:** 2-3 hours for all searches + outreach
**First responses expected:** Within 4-12 hours

### STEP 2: Email & Warm Outreach (1-2 hours)
**Secondary channel, higher conversion if warm**

**Warm referrals:**
- Ask 2-3 people on team for introductions to:
  - 1-2 Saudi legal/gov tech CTOs they know
  - 1-2 Arabic AI startup founders
  - 1-2 Western ML engineers from conferences/open-source

**Cold email:**
- If you have access to startup accelerator alumni lists (MEVP, AstroLabs), send templated emails
- Follow up 24 hours later
- Expected: 1-2 confirmations from 10+ cold emails

**Timeline:** 1-2 hours for outreach + follow-up
**Response window:** 24-48 hours

### STEP 3: Community & Social (30-45 min, optional)
**Lowest priority, but helps if falling short**

**Twitter/X:**
- Search #ArabicAI, #LLM, #GPUCompute hashtags
- Reply to relevant recent posts with recruitment pitch
- DM promising engineers directly

**Product Hunt / Hacker News:**
- Find threads on GPU marketplaces or LLM inference
- Post thoughtful comment with recruitment link

**Expected:** 0-2 confirmations (low conversion)

## Confirmation Workflow

**For each "YES" response:**

1. **Send confirmation email** (within 2 hours)
   - Zoom link + password
   - Session time (show 2 timezone options)
   - Consent form link
   - Pre-session survey link
   - Tech requirements (mic, camera, 10+ Mbps internet)

2. **Update tracker** (docs/ux/phase1-recruitment-tracker.md)
   - Change participant status from 🔴 to 🟢
   - Log name, email, company, session time
   - Mark as "Confirmed"

3. **Reminder emails**
   - 3 days before: Send consent form + survey
   - 24 hours before: Time zone reminder + tech check
   - Morning of: Final confirmation + Zoom link re-sent

## Contingency Plan

**If <5 confirmations by 3/24 EOD:**

Option 1: Run MVP testing (4 participants instead of 5-8)
- Still get directional feedback
- Less robust analysis, but acceptable for Phase 1

Option 2: Extend recruitment to 3/25 morning
- Run some sessions same day (risky, rushed)
- Compress timeline (fewer breaks between sessions)

Option 3: Defer to next week
- Full recruitment restart
- Delays Phase 1 go/no-go decision

**Recommendation:** Aim for 5-8. If 3-4 confirmed by 3/24 EOD, proceed with MVP (Option 1).
