## DCP Image Style Guide

### Core Visual Language
- **Primary palette:** Deep navy `#0D1B2A`, teal `#00E5C8`, orange `#FF6B00`
- **Aesthetic:** Dark tech with Arabic cultural warmth — NOT cold Silicon Valley minimalism
- **Settings:** Saudi Arabia (Riyadh skyline, desert, NEOM aesthetic), GPU hardware, data centers
- **People:** Diverse Middle Eastern representation, women in tech contexts, authentic not stock

### Prompt Templates

**Social media header (LinkedIn/Twitter):**
`Dark navy background #0D1B2A, teal and orange gradient light rays, GPU server hardware in foreground, Arabic calligraphy-inspired geometric patterns, cinematic lighting, 16:9, hyperrealistic, professional tech brand photography`

**Blog/article header:**
`Abstract GPU compute visualization, dark navy background, glowing teal data flow lines, orange energy nodes, infinity symbol motif, minimalist, 16:9, digital art`

**Provider hero image:**
`Saudi man in traditional thobe working on computer setup with multiple GPUs, modern apartment in Riyadh, warm lighting, teal screen glow, authentic, not stock photo style`

**Arabic AI concept:**
`Arabic calligraphy text glowing in teal and orange on dark navy background, AI neural network overlay, beautiful, editorial, tech magazine style`

### What to AVOID
- Generic Silicon Valley/Western stock imagery
- Blue-and-white color schemes (that's OpenAI/Meta)
- Robotic arms and circuit boards without cultural context
- White backgrounds (never — always dark navy)
