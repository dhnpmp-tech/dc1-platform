**Announcing DCP: Saudi Arabia's First Decentralized GPU Compute Marketplace**

We're building the infrastructure for AI compute in Saudi Arabia — and today we're opening DCP to the public.

**What is DCP?**

DCP connects GPU owners with AI developers and researchers. If you have an NVIDIA GPU sitting idle, you can list it on DCP and earn SAR for every hour it runs a compute job. If you're building AI applications and need GPU access, DCP gives you LLM inference, image generation, and model training via a simple REST API — billed in SAR, running on Saudi hardware.

**Why this matters for Saudi Arabia:**

The AI revolution requires compute. Today, Saudi ML engineers route their workloads through AWS, Azure, and Google Cloud — paying in USD, dealing with data residency concerns, and competing for capacity with global demand. DCP builds that capacity locally, on hardware already in the Kingdom, priced and paid in SAR.

**For GPU Owners (Providers):**

Register your NVIDIA GPU → install a lightweight daemon → earn SAR automatically. The daemon runs in the background; jobs execute in isolated Docker containers; your machine and data stay fully under your control. RTX 4090 at 15 SAR/hr = up to 360 SAR/day at full utilization.

Weekly IBAN payouts. Minimum 50 SAR. No crypto.

**For Developers and ML Engineers (Renters):**

Submit LLM inference jobs via REST API or our Python/JS SDKs. Browse the GPU marketplace, pick your compute tier, and pay per job in SAR. The VS Code extension (Ctrl+Shift+V) connects your editor directly to a live vLLM inference server running on DCP.

**PDPL Compliance:**
All compute runs on hardware in Saudi Arabia. Job data is ephemeral and never leaves the Kingdom.

**Where we are:**
- Provider registration: live at dcp.sa/provider/register
- Renter registration: live at dcp.sa/renter/register
- Full API documentation: dcp.sa/docs

We're looking for founding providers (GPU owners willing to list hardware and get locked-in founding rates) and early-access developers who want Saudi-hosted GPU compute for their projects.

If that's you — or if you know someone in KSA with idle GPU hardware — share this post and tag them.

🔗 dcp.sa

## Usage Notes

- **Format:** Long-form LinkedIn posts perform best with line breaks every 2–3 sentences — the formatting above is intentional
- **Audience targeting:** Post to Saudi tech groups (KACST alumni, Saudi Engineering Society, local AI meetup groups)
- **Best time:** Sunday or Monday morning (KSA), 9–11 AM AST
- **Visual:** Attach a graphic with the earnings table or a clean DCP logo + tagline card
- **Engagement:** In the first hour, respond to every comment — LinkedIn rewards early engagement with algorithmic reach
- **Tag strategy:** Avoid tagging competitors; do tag Saudi accelerators (SCAI, Monshaat) if there's a relationship
- **Cross-post:** Share the EN and AR versions as separate posts (not combined) for cleaner reach in each audience
